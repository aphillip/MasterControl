MasterControl Configuration Format


       

Attribute

Description

 

Configuration -

The root configuration node

password

The password used to restrict access to the configuration settings. If it is not blank, the user will be prompted to enter this value when File->Configuration is clicked in the menu.

 

TCP Server  -

List of tcp servers.

   

Server -

A tcp server accepting connections to process commands.

ip

The IP address of the server.

 
 

messageterminator

The series of characters that indicate to the server that the message is complete. The following special character sequences will be replaced in the message:

0x0 is replaced with null or zero.

0x0A is replaced with new line character.

0x0D is replaced with carriage return character.

This value will be appended to all Text mode messages sent to TCP servers. It is ignored for Hexadecimal messages.

 
 

name

A name for the server.

 
 

port

The network port to send the information to.

 
 

responsesize

The maximum size of the response in bytes.

 
 

timeout

The length of time to wait before giving up on establishing a connection.

 

User Interface -

User interface definition

alwaysontop

Whether the application will always remain the top-most window.  Can be True or False.

 

fontsize

The fontsize to use when rendering the interface.

 

fullscreen

Whether the application will occupy the entire screen or not.  Possible value of true or false. If true, the height and width attributes of the user interface settings will be ignored.

 

height

The height of the Master Control window.

 

restrictmouse

Whether the mouse should be restricted to the MasterControl window while the application is open.  Can be True or False.

 

title

The title of the Master Control window.

 

width

The width of the Master Control window.

 

Section -

A section on the User Interface

name

The name of the section.

 

Button -

A button in a section

color

The color of the button. Can be one of the following: red, green, blue, yellow, orange, lightorange, gray, darkgray, lightgray, pink, magenta, cyan, black, white.

 

label

The text to display on the button.

 

tooltip

The tooltip to display when the mouse hovers over the button for a short time.

 

exec       -

A button command that executes a program on the system.

command

The command/executable to be executed.

 
 

wait

Whether or not Master Control should wait for the command to end before processing any further instructions. Posible value of true or false.

 

sendkeys -

A button command that sends a combination of key presses to the system.  This can be used to trigger Hot Key events.

keylist

A + separted list of keys to press and release (Eg. ctrl+alt+c). The keys will be pressed and released in the order the appear in the list. Only the main character of a key can be used. For example, using ! Will not yeild any actions. To send an ! character, shift+1 can be used. Functions keys are refered to as F1, F2, etc.  The following is a list of non charcter keys that may be sent: ctrl, alt, shift, enter, caps, esc, tab, backspace, delete, home, end, pageup, pagedown, up, down, left, right, insert, numlock.

 
 

presstime

The amount of time in milliseconds to wait before the keys are released. Minimum value is 200.  Master Control will not process any commands until the presstime has elapsed.

 

tcpsend  -

A button command to send a value to a TCP Server.

readresponse

Whether or not Master Control should read a response from the target server(s).  Can be True or False. Some devices send a reponse after receiving a command and ignore further commands until it is read.  On the other hand, some devices don’t send a response, and trying to read one from the device will stall MasterControl.

 
 

target

The server to send the TCP command to. Multiple targets can be specified by separating the targets with a comma. No additional spaces should be used in the list. For example, two select targets “Left TV” and “Right TV”, the value entered should be:

 

Left TV,Right TV

 

Sending unknown commands to a target may cause MasterControl to stall.

 
 

value

The value to send to the TCP server.

 
   

valueformat

The format of the value that gets sent to the TCP servers.  Can be “Text” or “Hexadecimal”. For Hexadecmial, only hexidecimal characters are valid.

 

wait        -

A button command that halts program executes for a specified amount of time.  This is sometimes necessary to give deveices time to respond to the message sent.

time

The amount of time to wait.

 

At Exit           -

A set of commands to be executed when the program is exited.  The available child nodes of this node are the same as the child nodes for the button node.

enabled

Whether or not this feature is enabled.  Possible value of True or False. When True, the commands specified will be executed.
